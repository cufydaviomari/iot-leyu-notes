# iot-leyu-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面通常记录了一些零散的笔记、技术演示、工具页面或学习过程中的练习成果。每个 HTML 文件均可独立查看，不依赖于任何特定后端服务或域名。

该仓库的主要目的是提供一个结构化的存储空间，方便随时查阅、分享以及维护历史版本。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 可选：仓库首页导航
├── page-1.html        # 示例页面
├── page-2.html
└── ...
```

- 仓库根目录下直接存放各个 HTML 文件。
- 如有需要，可以创建子目录归类多个相关页面（例如 `tools/`, `demos/` 等）。
- `index.html` 并非必需，但可用于提供一个简单的页面索引。

## 页面归档说明

- 每个 HTML 文件均为自包含页面，所有样式、脚本（如有）通常内嵌或引用自公共资源（如 CDN）。
- 文件名应尽量简洁、有意义，便于识别内容主题。
- 页面内容可能包括但不限于：
  - 技术笔记与备忘
  - 小工具或交互演示
  - 学习过程中的实验页面
- 这些页面不提供任何形式的商业服务或功能承诺，仅供学习和参考使用。
